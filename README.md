# 🛠️ Fundamentos do Git e GitHub

Este repositório foi criado para consolidar meus conhecimentos em controle de versão utilizando Git e a plataforma GitHub. Durante os estudos, explorei desde os comandos básicos até fluxos de trabalho essenciais para o desenvolvimento de software.

## 📖 O que foi praticado:

- **Configuração Inicial:** Configuração de usuário, e-mail e chaves SSH.
- **Fluxo de Trabalho Local:** Iniciação de repositórios (`git init`), verificação de status (`status`) e histórico de commits (`log`).
- **Gerenciamento de Alterações:** Uso de `git add` (staging area) e `git commit` com mensagens claras.
- **Trabalho Remoto:** Conexão com repositórios remotos (`git remote`), envio de dados (`git push`) e atualização local (`git pull`).
- **Navegação no Tempo:** Uso de `git checkout` e `git restore` para gerenciar versões de arquivos.

## 🚀 Tecnologias Utilizadas

* **Git**: Sistema de controle de versão distribuído.
* **GitHub**: Plataforma de hospedagem de código e colaboração.
